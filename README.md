# 📊 Exam Mark Prediction using Multiple Linear Regression

🚀 A Machine Learning project that predicts student exam marks based on multiple factors like study hours, sleep, and previous performance using Multiple Linear Regression.

---

## 🎯 Project Overview

This project uses a supervised learning algorithm to analyze the relationship between multiple input variables and predict the final exam score.

📌 Built with:
- Python 🐍
- Scikit-learn 🤖
- Pandas 📊
- Streamlit 🌐

---

## 🧠 Problem Statement

Predict student performance (marks) using key influencing factors such as:
- 📚 Study Hours
- 😴 Sleep Hours
- 📝 Previous Scores

---

## 📐 Model Used

Multiple Linear Regression

Formula:
y = b0 + b1x1 + b2x2 + b3x3 + ... + bnxn

Where:
- y → Predicted Marks
- x → Input Features
- b → Coefficients
